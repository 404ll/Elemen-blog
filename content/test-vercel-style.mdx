# Vercel 博客样式测试

这篇文章展示了按照 Vercel 博客示例优化后的渲染效果。

## 主要改进

我们对博客渲染系统进行了以下改进：

1. **代码高亮** - 使用 `rehype-pretty-code` 和 `shiki` 提供美观的语法高亮
2. **MDX 组件** - 优化所有 MDX 组件的样式，使其符合 Vercel 设计规范
3. **代码块复制** - 改进的复制功能，支持语言标签显示
4. **整体排版** - 更好的阅读体验和视觉层次

## 代码示例

### JavaScript/TypeScript

下面是一个 React 组件示例：

```tsx
import { useState } from 'react'

interface TodoProps {
  title: string
  completed: boolean
}

export function TodoItem({ title, completed }: TodoProps) {
  const [isCompleted, setIsCompleted] = useState(completed)

  return (
    <div className="flex items-center gap-2">
      <input
        type="checkbox"
        checked={isCompleted}
        onChange={() => setIsCompleted(!isCompleted)}
      />
      <span className={isCompleted ? 'line-through' : ''}>
        {title}
      </span>
    </div>
  )
}
```

### Python 示例

```python
def fibonacci(n):
    """计算斐波那契数列的第 n 项"""
    if n <= 1:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

# 生成前 10 个斐波那契数
for i in range(10):
    print(f"fibonacci({i}) = {fibonacci(i)}")
```
