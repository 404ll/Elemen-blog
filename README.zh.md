## 🚀 关于本博客

这是一个融合 **Web2 前端工程** 与 **Web3 技术** 的个人技术空间，基于最新的现代技术栈构建：

- **Next.js App Router**
- **TypeScript**
- **TailwindCSS**
- **MDX 内容系统**

博客内容涵盖前端工程、区块链技术、智能合约、Move/Solidity、UI/UX，以及我的组件库。

只要我对某件事好奇 → 我就会构建它、记录它。
